# Page Builder Assets

Store Page Builder screenshots in these folders:

- `sections`: reusable section screenshots
- `pages`: full-page screenshot references
- `thumbnails`: generated or curated preview images

Use PNG, JPG, JPEG, or WebP files with clear lowercase filenames and hyphens.
Keep these assets inside this tool folder, not in the generic public folder.
Asset discovery and rendering will be implemented in the next cycle.
